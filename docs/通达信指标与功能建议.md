# 通达信指标在 Rox 中的用法与功能建议

## 一、通达信指标在 Rox 里可以用在哪里

您提供的几类指标与 Rox 现有能力对应关系如下。

### 1. 游资暗盘 / 暗盘资金 hf1.0

- **Rox 现状**：已有 `app/analysis/hf_indicator.py`（游资净买、MX/MXA 等）、`app/analysis/hot_money.py`，专业版信号分析里会用到类似逻辑。
- **建议用法**：
  - **ROX 3.0 专业版 → 信号分析**：在「信号分析」接口中继续作为「游资/暗盘」类信号，与亢龙有悔、仙人指路等一起参与多信号融合与买卖建议。
  - **个股诊断（1.0）**：若在诊断里增加「资金/游资」维度，可在这里调用游资暗盘、暗盘资金的结果做简要提示（如「游资净买上穿/下穿」）。
  - **K 线副图**：若后续在专业版 K 线图增加副图指标，可把「游资净买 / JJ / XJ」等画成副图曲线。

### 2. 亢龙有悔（看龙有悔）

- **Rox 现状**：已有 `app/analysis/kang_long_you_hui.py`，实现前高、倍量、突破、启明/揽月线、亢龙有悔文字等逻辑，并在专业版信号分析中使用。
- **建议用法**：
  - **专业版信号分析**：继续作为独立信号源，输出买/卖或观望结论。
  - **选股**：若做「突破前高 + 倍量」类选股，可直接复用该模块的 `signal_xg`、`signal_zhong_dian` 等条件。

### 3. 仙人指路 2.0

- **Rox 现状**：有「金钻趋势」、XMA、回调买、DDX 等思路的模块（如 `app/analysis/precise_trading.py` 等），与仙人指路 2.0 部分重合。
- **建议用法**：
  - **专业版**：在信号分析或选股中增加「金钻趋势 + 回调买」「金钻起涨」等条件，与现有精确买点一起输出。
  - **K 线主图**：若支持自定义主图指标，可把「金钻趋势 / 金牛2」画在主图上。

### 4. 机构操盘 3.0

- **Rox 现状**：专业版有多信号融合、仓位建议；红色持股/青色观望、短买/白色离场等状态可与「机构操盘」的持股/观望/离场对应。
- **建议用法**：
  - **专业版**：在综合信号或策略模板中，把「红色持股 / 青色观望 / 短买 / 白色离场」做成状态输出或仓位建议（例如对应 334 中的底仓/律动/预备）。
  - **阻力/支撑**：指标中的「今日阻力、今日支撑、明日突破、明日反转」可单独作为接口字段，供前端或复盘使用。

### 5. 寻龙诀选股器

- **Rox 现状**：暂无与「寻龙诀」完全一致的选股公式。
- **建议用法**：
  - **选股 API**：新增选股接口（如 `/api/strategy/screen` 或专业版选股），把寻龙诀条件（前高、倍量、突破、X_19/X_24、涨停等）用 Python 实现，返回代码列表。
  - **专业版选股页**：在 UI 上提供「寻龙诀」预设，一键执行并展示结果。

### 6. 暗盘资金 hf1.0（分档资金 + 调整幅度）

- **Rox 现状**：暗盘资金涉及 BIGBUYMONEY1 等分档数据，Rox 若已接 Level2 或同类数据，可在服务端算「暗盘资金」；否则需先接数据源。
- **建议用法**：
  - **有分档数据时**：在专业版资金分析或信号分析中增加「暗盘资金」曲线/强度，与游资暗盘一起用。
  - **暂无分档数据时**：先用现有主力/北向等替代，待数据到位再接入该公式。

## 二、交易复盘（每个周期 AI 帮忙复盘）

- **已做**：交易板块增加了「AI 复盘」入口：「生成本周复盘」「生成本月复盘」按钮，以及展示复盘的文案说明；后端新增 `GET /api/trade/review?period=week|month`，按周/月统计近期成交笔数并返回简要摘要。
- **后续**：可在该接口内接入 LLM，根据近期成交列表（代码、买卖、价格、盈亏等）生成「得失分析 + 策略建议」的复盘文本；前端已预留展示区域，接口返回 `summary` 即可显示。

## 三、宏观经济仪表盘

- **作用**：观察 PMI、CPI/PPI、M2、国债收益率、汇率等，判断经济周期与政策环境，为**仓位与行业配置**提供大方向（不是直接给出买卖点）。
- **当前操作逻辑**（已写在宏观页顶部）：
  1. 制造业 PMI &gt;50 为扩张，&lt;50 为收缩；
  2. CPI/PPI 看通胀压力；
  3. M2/社融看流动性；
  4. 10 年国债与汇率影响资金成本与外资流向；综合偏暖可适当提高权益仓位，偏冷或高通胀偏向防御或观望。
- **是否有必要单独说明**：有必要；已在宏观板块顶部增加「板块作用」和「当前操作逻辑」说明。
- **专业建议**：
  - 可增加：社融同比、工业增加值、失业率、美联储利率/美债收益率（若做跨境或外资流向）。
  - 若数据源支持，可增加简单「宏观状态」标签（如「复苏 / 滞胀 / 衰退」）和一两句操作建议，便于非专业用户理解。

---

## 四、汇率是否单独建板块

- **建议**：当前在宏观仪表盘内展示「美元/人民币」即可，不必单独建「汇率」板块。
- **若后续**：增加多币对（如 EUR/CNY、一篮子）、历史曲线、与北向资金联动等，再单独建「汇率」子板块或独立页更合适。

## 五、每周推荐还需增加什么

- **334 柱状图含义**（已写在每周推荐页）：  
  柱状图表示 **334 仓位法则** 的三类仓位占比：**底仓 30%**、**律动 30%**、**预备队 40%**（不是「周一至周五」）。底仓长期持有，律动做波段降成本，预备队用于加仓或应对风险。图表已改为「底仓30% / 律动30% / 预备40%」三根柱，并配有简短说明。
- **可增加内容**：
  - **本周宏观简要**：一两句当前 PMI/流动性结论，与宏观页一致。
  - **仓位建议**：例如「建议总仓位 X%」「底仓/律动/预备占比」。
  - **行业或风格**：当前偏价值/成长、偏哪些行业（若后端有模型可输出）。
  - **推荐理由结构化**：每只推荐标的除「理由」外，可增加「技术/资金/基本面」标签或评分，便于和个股诊断、专业版信号对齐。
