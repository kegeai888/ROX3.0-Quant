# ROX Quant 如何启动

## 一、推荐：用浏览器打开（最简单）

在终端里执行（**一定要先进入项目目录**）：

```bash
cd /Users/mac/Downloads/rox3.0
python3 -m uvicorn app.main:app --host 127.0.0.1 --port 8008
```

看到类似 `Uvicorn running on http://127.0.0.1:8008` 后，**用浏览器打开**：

- 经典版（市场看板、宏观、每周推荐等）：**http://127.0.0.1:8008**
- 专业版：**http://127.0.0.1:8008/pro**

不要关掉终端，关掉后服务就停了。

---

## 二、用脚本启动（端口 8081）

```bash
cd /Users/mac/Downloads/rox3.0
chmod +x start_server.sh   # 只需执行一次，赋予执行权限
./start_server.sh
```

然后在浏览器打开：**http://127.0.0.1:8081**

---

## 三、桌面窗口版（会弹出一个软件窗口）

```bash
cd /Users/mac/Downloads/rox3.0
chmod +x start_rox.sh   # 只需执行一次
./start_rox.sh
```

或：

```bash
cd /Users/mac/Downloads/rox3.0
python3 rox_desktop.py
```

会先启动后端（端口 **8008**），再弹出一个桌面窗口，里面就是 ROX Quant 界面。

---

## 打不开时请检查

1. **是否在项目目录下执行**  
   一定要先执行 `cd /Users/mac/Downloads/rox3.0`，再执行上面的命令。

2. **端口被占用**  
   - 方式一用的是 **8008**，方式二用的是 **8081**，桌面版用的是 **8008**。  
   - 若提示端口已被占用，可换一个端口，例如：
     ```bash
     python3 -m uvicorn app.main:app --host 127.0.0.1 --port 9000
     ```
     然后浏览器打开 **http://127.0.0.1:9000**。

3. **依赖未安装**  
   在项目目录下执行：
   ```bash
   pip3 install -r requirements.txt
   ```
   若用虚拟环境：
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
   然后再用上面任一方式启动。

4. **终端里的报错**  
   把终端里最后一屏的完整报错复制下来，便于排查（例如 uvicorn 报错、import 报错等）。
